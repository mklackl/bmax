## Task: Technical SEO Audit

Audit the project's SEO setup:

1. **On-Page SEO**
   - Title tags: unique, under 60 chars, keyword-rich
   - Meta descriptions: unique, under 160 chars, compelling
   - H1 tags: one per page, descriptive
   - Image alt text present
   - Internal linking structure

2. **Technical SEO**
   - Sitemap.xml exists and is valid
   - robots.txt configured correctly
   - Canonical URLs set
   - 404 page exists
   - Redirects working (no chains)
   - Page speed (Core Web Vitals targets)

3. **Structured Data**
   - JSON-LD schema markup (Organization, Product, FAQ as applicable)
   - Open Graph tags for social sharing
   - Twitter Card tags

4. **Indexing**
   - Google Search Console connected
   - Key pages are indexable (not accidentally noindexed)
   - URL structure is clean and semantic

5. **Content SEO**
   - Landing page targets primary keyword
   - Blog/docs set up for content marketing
   - Internal linking between related pages

Output findings and fixes as `_bmad-output/planning-artifacts/seo-audit.md`.
