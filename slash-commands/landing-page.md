## Task: Landing Page Planning

Design the landing page structure and copy:

1. **Hero Section**
   - Headline (clear value proposition, under 10 words)
   - Subheadline (supporting detail, 1-2 sentences)
   - Primary CTA (what action? what copy on the button?)
   - Hero visual concept

2. **Social Proof** (if available)
   - Testimonials, user count, logos, ratings
   - Fallback if no social proof yet (beta invitation framing)

3. **Feature Sections** (3-5 features)
   - Each: icon/visual + headline + 2-sentence description
   - Focus on benefits, not features

4. **How It Works** (3-4 steps)
   - Simple numbered steps
   - Reduce perceived complexity

5. **Pricing Section**
   - Inline pricing or link to pricing page
   - Highlight recommended tier

6. **FAQ Section**
   - 5-8 questions addressing objections
   - Include: pricing, data security, cancellation, support

7. **Final CTA**
   - Repeat primary CTA
   - Urgency or scarcity (if authentic)

8. **SEO Requirements**
   - Target keyword for this page
   - Meta title and description
   - Schema markup recommendations

Output as `_bmad-output/planning-artifacts/landing-page.md`.
